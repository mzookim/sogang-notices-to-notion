# Contributing

이 저장소는 개인 포트폴리오 성격의 프로젝트입니다. 이슈/PR은 환영하지만, 모든 제안이 반영되거나 유지보수되는 것을 보장하진 않습니다.

## 이슈
- 재현 가능한 버그인 경우, 재현 방법과 로그를 포함해 주세요.
- 보안 취약점은 공개 이슈가 아닌 `SECURITY.md`를 참고해 주세요.

## PR
- 기능 추가는 작은 단위로 나누어 주세요.
- 변경 사유와 영향 범위를 설명해 주세요.
- 테스트가 없다면, 최소한 실행 절차를 적어 주세요.

## 커밋 메시지
- `type(scope): subject` 형식을 권장합니다.
